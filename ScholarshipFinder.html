Scholarship Finder

Built a scholarship discovery and application-assistance system combining large-scale web scraping, structured data management, AI matching, document parsing, and essay-generation workflows.

Web Scraping Pipeline

Created custom scrapers for sources such as Scholarships.com and Scholarships360, extracting scholarship titles, descriptions, award amounts, deadlines, eligibility requirements, application links, source URLs, and category metadata into normalized JSON datasets.

Data Management

Grouped scholarships by major, career interest, grade level, location, demographic eligibility, financial need, merit criteria, essay requirement, deadline window, and award type so the AI can search targeted categories before ranking matches.

Vector Matching

Used vector embeddings of scholarship keywords, descriptions, eligibility rules, and user profile details so the system can find semantically similar scholarships instead of relying only on exact keyword overlap.

Agent Tools

Designed tool-based AI workflows for scholarship_search, category_filter, eligibility_checker, document_parser, document_search, and essay_drafter to connect scholarship retrieval with application support.

Document Parsing

Uploaded PDFs, resumes, transcripts, essays, and activity lists are parsed into clean text, split into meaningful chunks, converted into embeddings, and indexed in a vector store for fast retrieval from large documents.

RAG Essay Support

The chatbot uses retrieval-augmented generation to repurpose uploaded documents into tailored scholarship essays, retrieving relevant examples, checking eligibility context, and drafting focused responses aligned with each prompt.
